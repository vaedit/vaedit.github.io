如果遇到该报错

请在两台服务器上都执行yum install rsync -y,或者源码安装完后，做一个软连接，让rsync命令能在PATH路径中找到

注意点

1、如果使用命令时只指定源而不指定目标，仅会将源以列表的形式显示而不同步

2、rsync命令使用中，如果源参数的末尾有斜线，只会复制指定目录的内容，而不复制目录本身，没有斜线，则会复制目录本身，包括目录
